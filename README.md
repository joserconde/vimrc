# vimrc

My vimrc files

## Installation

* Clone the repository into the ~/.vim/ folder.
* Create a symbolic link from ~/.vimrc to ~/.vim/vimrc
* Initialice the git submodule and update it.

```bash
ln -s ~/.vimrc ~/.vim/vimrc
cd ~/.vim/vimrc
git submodule init
git submodule update
```

Enjoy! :blush:
